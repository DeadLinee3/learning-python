## 1. Базовые команды

**print** — вывод текста или данных на экран.  
Пример:
```python
print("Привет, мир!")  # → Привет, мир!
```

**input** — ввод данных пользователем.  
Пример:
```python
name = input("Введите имя: ")
print("Привет,", name)
```

**len** — возвращает длину объекта (строки, списка и т.д.).  
Пример:
```python
name = "Python"
print(len(name))  # → 6
```

## 2. Математические операторы

| Символ | Название | Описание | Пример |
|--------|----------|----------|--------|
| `+` | Плюс | Сложение | `3 + 2 → 5` |
| `-` | Минус | Вычитание | `5 - 2 → 3` |
| `*` | Звёздочка | Умножение | `4 * 2 → 8` |
| `/` | Слэш | Деление (результат с дробью) | `5 / 2 → 2.5` |
| `//` | Двойной слэш | Целочисленное деление | `5 // 2 → 2` |
| `%` | Процент | Остаток от деления | `5 % 2 → 1` |
| `**` | Две звёздочки | Возведение в степень | `2 ** 3 → 8` |

## 3. Логические конструкции

**if** — если, проверяет условие.  
**elif** — иначе если (второе условие).  
**else** — иначе, выполняется, если все условия выше не сработали.

Пример:
```python
x = 5
if x > 0:
    print("Положительное число")
elif x == 0:
    print("Ноль")
else:
    print("Отрицательное число")
```

## 4. Символы и операторы сравнения

| Символ | Значение | Пример |
|--------|----------|--------|

Циклы while и for, а также continue / break
1) Быстрое сравнение: когда while, а когда for?

while — когда заранее не знаем, сколько раз повторять. Часто вместе с вводом пользователя, «пока не…».

for — когда знаем количество повторений или перебираем элементы коллекции (диапазон чисел, список, строку).

Примеры:

# while: спрашиваем пока не введут 'exit'
while True:
    cmd = input(">>> ")
    if cmd == "exit":
        break
    print("Команда:", cmd)

# for: 5 раз что-то делаем
for i in range(5):
    print("Итерация:", i)

2) Цикл for и range()

range() создаёт последовательность чисел (правая граница не включается):

range(stop) → от 0 до stop-1

range(start, stop) → от start до stop-1

range(start, stop, step) → с шагом step (может быть отрицательным)

Примеры:

for i in range(5):         # 0,1,2,3,4
    print(i)

for i in range(2, 6):      # 2,3,4,5
    print(i)

for i in range(10, 0, -2): # 10,8,6,4,2
    print(i)


Перебор коллекций:

for ch in "abc":
    print(ch)  # a, затем b, затем c

nums = [10, 20, 30]
for n in nums:
    print(n)

3) continue — пропустить остаток итерации и перейти к следующей

Что делает: как только интерпретатор встречает continue внутри цикла, он прекращает выполнять оставшийся код в текущем круге и сразу начинает следующий круг.

Пример с for
for i in range(1, 6):
    if i == 3:
        # пропускаем печать для i=3
        continue
    print(i)
# Вывод: 1, 2, 4, 5

Пример с while
i = 0
while i < 5:
    i += 1
    if i == 3:
        continue  # пропустит print для i=3
    print(i)
# Вывод: 1, 2, 4, 5


Зачем нужно: удобно для валидации ввода или пропуска «неподходящих» случаев без лишних else.

4) break — досрочно выйти из цикла

Что делает: полностью останавливает цикл (в отличие от continue, который просто перескакивает на следующую итерацию).

Пример:

for i in range(1, 100):
    print(i)
    if i == 5:
        break   # цикл завершится на 5
# Вывод: 1,2,3,4,5

5) Валидация ввода с continue (частая реальная схема)

Задача: спросить число от 1 до 10. Если ввели что-то вне диапазона — сообщить и спросить снова. Если нормальное — вывести квадрат и завершить.

while True:
    num = int(input("Введи число от 1 до 10: "))

    # проверка диапазона
    if num < 1 or num > 10:
        print("Вне диапазона, попробуй снова!")
        continue  # прыгаем к началу цикла, просим ещё раз

    # сюда попадём только если число корректное
    print(f"Квадрат числа: {num * num}")
    break  # завершаем цикл

6) Типичные ошибки и как их избегать

Забыли менять счётчик в while → бесконечный цикл.

i = 0
while i < 3:
    print(i)
    i += 1  # не забыть!


Путаем = и ==:
= — присваивание, == — сравнение.

Неправильные отступы после строк с : (if, elif, else, for, while).

Вычисления вне {} в f-строке. Любое значение/выражение для подстановки — в фигурные скобки:

print(f"{a} + {b} = {a + b}")

7) Мини-шпаргалка (как думать)

«Что повторять?» → это тело цикла (while/for).

«Когда остановиться?» → break.

«Что пропустить и начать заново круг?» → continue.

«Сколько раз повторять известно заранее?» → for + range.

«Повторять пока условие/событие?» → while.

8) Мини-практика (для закрепления)

Сумма только положительных чисел: пять раз спросить число; если <= 0 — continue, иначе прибавить к сумме; вывести итог.
