## 1. Цикл `while` — «пока условие верно»
**Описание:** Выполняет блок кода, пока условие истинно (`True`).  
Если условие становится ложным (`False`), цикл прекращается.

**Пример:**
```python
i = 0
while i < 5:
    print(i)
    i += 1
