# Errors — Частые ошибки Python

| Ошибка | Что значит | Пример и решение |
|--------|------------|------------------|
| `SyntaxError` | Ошибка в синтаксисе | Пропущено `:` → `if True` ❌ → `if True:` ✅ |
| `IndentationError` | Неправильные отступы | Код без отступа в блоке `if` |
| `NameError` | Переменная не найдена | Использована `y`, но она не определена |
| `TypeError` | Неподходящий тип данных | `"5" + 3` ❌ → `int("5") + 3` ✅ |
| `ValueError` | Неправильное значение для типа | `int("abc")` ❌ |
