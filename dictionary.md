---

## 1. Основные команды

| Команда | Описание | Пример |
|----------|----------|--------|
| `print()` | Вывод текста или данных на экран | `print("Привет, мир!")` |
| `input()` | Ввод данных от пользователя | `name = input("Введите имя: ")` |
| `len()` | Длина объекта (строки, списка) | `len("Python") → 6` |
| `type()` | Узнать тип данных | `type(42) → <class 'int'>` |
| `int()` | Преобразование в число | `int("5") → 5` |
| `str()` | Преобразование в строку | `str(5) → "5"` |
| `float()` | Преобразование в число с точкой | `float("5.2") → 5.2` |

---

## 2. Символы и операторы

| Символ | Название / Описание | Пример |
|--------|----------------------|--------|
| `=` | Присваивание (запись значения в переменную) | `x = 5` |
| `==` | Сравнение (проверка равенства) | `if x == 5:` |
| `!=` | Не равно | `if x != 3:` |
| `<, >, <=, >=` | Больше, меньше, и их варианты | `if x >= 10:` |
| `:` | Начало блока кода | `if True:` |
| `#` | Комментарий | `# Это комментарий` |
| `""` или `''` | Строка | `"Hello"` |
| `()` | Круглые скобки: вызовы функций, кортежи | `print("Hi")` |
| `[]` | Квадратные скобки: списки | `[1, 2, 3]` |
| `{}` | Фигурные скобки: словари или множества | `{"a": 1}` |
| `+` | Сложение (или конкатенация строк) | `5 + 3 → 8` |
| `-` | Вычитание | `5 - 2 → 3` |
| `*` | Умножение | `4 * 2 → 8` |
| `/` | Деление (результат всегда float) | `5 / 2 → 2.5` |
| `//` | Целочисленное деление | `5 // 2 → 2` |
| `%` | Остаток от деления | `5 % 2 → 1` |
| `**` | Возведение в степень | `2 ** 3 → 8` |

---

## 3. Ветвления (`if / elif / else`)

| Конструкция | Описание | Пример |
|-------------|----------|--------|
| `if` | Проверка условия | `if x > 0: print("Положительное")` |
| `elif` | Доп. условие (проверяется, если предыдущее не выполнено) | `elif x == 0:` |
| `else` | Выполняется, если все выше были False | `else: print("Отрицательное")` |

**Пример:**
```python
x = int(input("Введите число: "))

if x > 0:
    print("Положительное число")
elif x == 0:
    print("Ноль")
else:
    print("Отрицательное число")
